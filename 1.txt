Тема 3. Системи контролю версій та їх використання.

Лабораторна робота 7

Тема: Конфігурація Git та створення Git репозиторію. Додавання файлів до Git репозиторію Git та фіксування змін

Мета: Ознайомитися з основними налаштуваннями Git, навчитися ініціалізувати новий репозиторій Git, додавати файли до репозиторію та фіксувати зміни за допомогою комітів. У результаті виконання лабораторної роботи студент має засвоїти основи роботи з системою контролю версій Git, а також зрозуміти принципи командної роботи над проєктами.
